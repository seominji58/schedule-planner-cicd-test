[기획 요약] “내 일정을 부탁해” – 기업용 프로젝트 일정 관리 시스템

1. 제품 비전

‘내 일정을 부탁해’는 기업 환경에서의 프로젝트 중심 일정 관리를 지원하는 웹 애플리케이션입니다.
개인-팀-부서-프로젝트 전반의 일정을 통합하여 구성원 간 일정 공유와 협업을 가능하게 하고,
AI 기반 역산 스케줄링(마감일 기준 일정 자동 생성) 기능을 통해 업무 효율성을 높입니다.

2. 타깃 사용자

- 팀 단위 협업이 중요한 기업/스타트업
- 프로젝트 기반으로 일정을 진행하는 기획/개발/디자인 부서
- 업무 마감 기한을 중심으로 일정 조정이 필요한 PM, PO
- 부서/팀 간 공유 일정 관리가 필요한 중간 관리자

3. 핵심 기능 요약

🔹 멀티 계층 일정 구조
- 개인 일정: 개인이 등록한 일정 (AI 분석 대상)
- 팀 일정: 팀 구성원이 공유하는 일정
- 부서 일정: 부서 단위 공통 일정
- 프로젝트 일정: 마일스톤 기반 전체 일정

🔹 AI 기반 역산 일정 관리
- 마감일 기준 일정 자동 생성
- 리소스 충돌 자동 감지
- 우선순위 기반 일정 재배치

🔹 구성원 간 일정 공유 및 접근 제어
- 일정별 보기/편집 권한 설정 (팀, 부서, 관리자)
- 역할 기반 뷰 필터링 (개인/팀/부서/전체)
- 알림 및 마감 리마인더 전송 (이메일, 앱, 슬랙 등 확장 가능)

4. 기대 효과

- 업무 중복 방지: 프로젝트 단위 일정 공유로 일정 간소화
- 마감일 관리 용이: 역산 일정 자동 생성으로 일정 누락 최소화
- 조직 내 가시성 향상: 개인 일정과 팀 일정을 통합해서 한눈에 확인
- 일정 기반 협업 효율 증대: 할일 기반이 아닌 일정 기반 협업 문화 정착

5. 다음 추천 작업

- 와이어프레임 구성: 개인/팀/부서/프로젝트 뷰 UX 흐름 설계
- 데이터 모델 설계: Schedule, User, Role, Project 등의 ERD 구성
- API 명세서: 프로젝트/일정 CRUD, 권한 설정, 공유 API 정의
- 회사 중심 소개 페이지용 브랜딩 문구 정리
